## 📋 Sobre o Projeto

**Sabor da Angel** é um site completo para marmitaria com cardápio digital integrado. O projeto foi desenvolvido com foco em performance, responsividade e experiência do usuário, permitindo que clientes visualizem o cardápio, filtrem produtos por categoria e façam pedidos diretamente pelo WhatsApp.

### ✨ Funcionalidades

- 🏠 **Landing Page** — Seção hero com animações, estatísticas e call-to-action
- 🍽️ **Cardápio Digital** — Grid de produtos com filtros por categoria (Kits, Frango, Carne, Peixe, Vegano, Low Carb)
- 📱 **Pedido via WhatsApp** — Integração direta em cada produto do cardápio
- 💬 **Depoimentos** — Slider automático com avaliações de clientes
- 🚚 **Informações de Entrega** — Seção dedicada com detalhes de horário e agendamento
- 💳 **Formas de Pagamento** — Crédito, Débito, PIX, Dinheiro e Vale Refeição
- 📱 **100% Responsivo** — Adaptado para desktop, tablet e mobile
- 🎨 **Animações** — Scroll animations, partículas, floating cards e transições suaves
- 🔝 **Botão WhatsApp Flutuante** — Sempre visível para facilitar o contato

## 🚀 Como Usar

### 1. Clone o repositório
```bash
git clone https://github.com/jovemegidio/Marmitaria-Cardapio.git
```

### 2. Abra no navegador
Basta abrir o arquivo `index.html` no navegador. Não precisa de servidor!

### 3. Personalize
- **WhatsApp**: Substitua `5511999999999` pelo número real no `index.html`
- **Imagens**: Troque os SVGs em `img/` por fotos reais dos produtos
- **Preços**: Atualize os valores no `index.html`
- **Redes sociais**: Atualize os links no footer

## 📱 Responsividade

O site se adapta automaticamente a diferentes tamanhos de tela:

- **Desktop** (1200px+) — Layout completo em grid
- **Tablet** (768px - 1024px) — Grid adaptado, 2 colunas
- **Mobile** (< 768px) — Layout single column, menu hamburger
